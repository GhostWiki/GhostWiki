# GhostWiki

[GhostWiki](https://github.com/GhostWiki/GhostWiki) users can effortlesly store and manage files within nested folders


## üìö Project Definition

GhostWiki is an opinionated wiki system that provides a pleasant and simple out-of-the-box experience. New users to an instance should find the experience intuitive and only basic word-processing skills should be required to get involved in creating content on GhostWiki. The platform should provide advanced power features to those that desire it but they should not interfere with the core simple user experience.

GhostWiki is not designed as an extensible platform to be used for purposes that differ to the statement above.

In regard to development philosophy, GhostWiki has a relaxed, open & positive approach. At the end of the day this is free software developed and maintained by people donating their own free time.
